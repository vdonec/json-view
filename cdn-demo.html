json-view CDN Demo

Interactive demonstration of the @vdonec/json-view library loaded from CDN with all virtualization and navigation features

ℹ️ How it works
No build step required! This page loads json-view directly from jsdelivr.net, so you can copy-paste the CDN links into any HTML file and start using the library immediately.

The library is loaded as an ES module from CDN and exposed as window.jsonview for this page:

  • renderJSON(data, element, options) — Render JSON into the DOM
  • expand(node) — Expand all nodes
  • collapse(node) — Collapse all nodes
  • destroy(tree) — Clean up and remove from DOM
